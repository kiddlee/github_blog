date:   2016-01-14
description: How to upgrade php 5.6 in ubuntu14
categories: Linux
tags: php
---

```
sudo apt-get install software-properties-common
sudo add-apt-repository ppa:ondrej/php5-5.6
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install php5
```
